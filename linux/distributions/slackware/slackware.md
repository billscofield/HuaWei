## 安装步骤

1. 分区
    
    boot 分区

    / 分区

2. setup

    可以从第四个步骤开始

    1. TARGET:
        先选择 / 分区
        不用格式化
        
        然后选择 boot 分区, 挂载到 /boot
    
    2. 进入安装源
        
        选择 install from a slackware CD or DVD

    3. 选择安装包
